# src/model/

Gene annotation model: bidirectional Mamba tower with optional RNA-Seq cross-attention and HMM decoder.

## Contents

- `jax/` — JAX/Flax implementation for training. Modules: `SubbyModel`, `MambaTower`, `TrackEncoder`, `TrackCrossAttention`, `BidirectionalMamba`, `SelectiveSSM`. See `docs/api/model.md`.
- `webgpu/` — WebGPU inference implementation (planned).
- `hmm_decoder.json` — 27-state dual-strand Machine Boss transducer. 15 Tiberius-compatible emission/annotation labels. Generated by `scripts/generate_hmm_json.py`.

## Tests

`tests/test_model/` — shape, gradient flow, and weight sharing tests for all model components.
