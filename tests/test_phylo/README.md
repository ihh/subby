# tests/test_phylo/

Tests for the phylogenetic sufficient statistics library (`src/phylo/`).

- `test_pruning.py` — Felsenstein upward pass (inside algorithm)
- `test_outside.py` — Downward pass (outside algorithm)
- `test_eigensub.py` — Eigensubstitution accumulation (J matrices, eigenbasis projection, C accumulation, back-transform)
- `test_f81_fast.py` — O(CRA^2) F81/Jukes-Cantor fast path
- `test_hky85.py` — HKY85 model diagonalization and substitution matrices
- `test_components.py` — Tree construction, token-to-likelihood, branch masking
- `test_api.py` — Top-level API (`LogLike`, `Counts`, `RootProb`, `MixturePosterior`)
- `test_oracle.py` — Oracle vs JAX cross-validation (atol=1e-8)
- `golden/` — Deterministic JSON test fixtures generated by `scripts/generate_golden_tests.py`
