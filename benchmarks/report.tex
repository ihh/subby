\section{Methodology}

We benchmark two functions from the phylogenetic sufficient-statistics library:
\begin{itemize}
  \item \texttt{LogLike()}: upward pass only (Felsenstein pruning), computing per-column log-likelihoods.
  \item \texttt{Counts()}: full eigensubstitution pipeline (upward pass, downward pass, eigensubstitution accumulation, back-transform), computing expected substitution counts and dwell times.
\end{itemize}

Both functions are timed across a parameter grid:
\begin{itemize}
  \item Alphabet size $A \in \{4, 20, 64\}$
  \item Alignment width $C \in \{100, 1000, 10000\}$ columns
  \item Tree depth $R \in \{7, 15, 31\}$ nodes (balanced binary)
\end{itemize}

Input data are synthetic: Jukes--Cantor models with random balanced binary trees and random alignments with occasional gaps.
For JAX backends, a warmup call is made before timing to allow JIT compilation.
Each configuration is timed over multiple repetitions; we report mean $\pm$ standard deviation.
